# Hazelslider


## 1. Call the HazelSider

``` Javascript
new HazelSlider({
    container: '.hs-container',
    slideX: 6,
    slideY: 6,
    width: 960, // if you want to use % then you will use like '100%'. if you don't need % then just use number like 500
    //height: 600,
    imageHeightRatio: 50 // Default 46
});

```
